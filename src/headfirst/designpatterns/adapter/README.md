# 适配器模式
## 定义
将一个类的接口,转换成客户期望的另一个接口。适配器让原本接口不兼容的类可以合作无间。


## 场景
- 适配器工作起来就如同一个中间人,它将客户发出的请求转换成厂商类能理解的请求。
    - 例如: 电源适配器
- 用户调用适配器接口, 具体适配器实现适配器接口并持有被适配对象, 在用户想要的功能接口内部调用被适配对象的真正接口。
    - 比如用户想要一个方接口, 但实际上只有圆接口, 那可以用一个适配器, 用户调用适配器的方接口, 适配器内部调用真正的圆接口。
